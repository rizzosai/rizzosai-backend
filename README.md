# RizzosAI Backend System

Revolutionary "Next 5" commission system with Flask backend.

## Features
- Fair-share commission queue system
- Affiliate tracking and management
- Lead capture and follow-up
- Professional dashboard interface
- Zapier webhook integration
- Mobile-responsive design

## Deployment
Ready for Render deployment with production configuration.

## Local Development
```bash
pip install -r requirements.txt
python app.py
```

## Environment Variables
- PORT: Server port (default: 5000)
